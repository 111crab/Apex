# 技能编辑器愿景记录

*创建日期：2026-06-26*  
*状态：愿景记录，不进入当前实现。*

## 核心类比

技能编辑器可以类比为“播放 Montage 的编辑器界面”：
- 可以在细节面板配置技能参数。
- 可以在预览区域播放技能效果。
- 可以在编辑器中预览表现和关键时序。
- 点击 Play / 游玩游戏时，配置过的技能可以被角色实际使用。
- 已确认保存的技能以配置资产形式存储，下次启动项目仍可使用。

## 设计含义

这意味着技能编辑器不应该只是一个数据表编辑器，也不应该第一步就变成复杂图编辑器。更合理的方向是：
- 先有稳定的技能配置资产。
- 再有能读取配置并运行的通用运行时。
- 编辑器围绕这个配置资产提供细节面板、预览、验证、保存。
- 技能预览类似 Montage 预览：重点看动画、特效、命中时机、范围、目标、持续时间，而不是生成 C++ 代码。
