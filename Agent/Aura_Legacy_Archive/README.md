# Agent 协作区

这个目录用于存放 Aura 技能系统项目的协作文档。它不是游戏源码目录：运行时代码仍放在 `Source/`，UE 资产仍放在 `Content/`，规划、审查、交接、阶段总结放在这里。

## 目录说明

| 目录 | 用途 |
| --- | --- |
| `00_Coordination/` | 协作规则、任务池、决策记录、Git 工作流、Skill/MCP 目录。 |
| `01_ProjectAudit/` | 当前项目状态、代码结构审计、资产盘点等。 |
| `02_SkillSystem/` | 后续存放技能系统设计文档。Phase 1 开始时再创建。 |
| `03_SkillEditor/` | 后续存放技能编辑器设计文档。编辑器阶段再创建。 |
| `04_AgentHandoff/` | Codex 与 Claude 之间的计划、Prompt、实施报告。 |
| `90_References/` | 后续存放外部参考资料和评审。 |

## 现有参考文档
