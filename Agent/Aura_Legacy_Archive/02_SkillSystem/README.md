# 技能系统设计区

这里存放 Aura 技能系统本体的设计文档。

当前建议顺序：

1. `Skill_Mechanic_Catalog.md`  
   技能机制总目录：先尽量系统地整理值得抽象的技能机制全集。

2. `Skill_Mechanic_Taxonomy.md`  
   已确认的技能机制分类与第一批原型边界。

3. `Representative_Skill_Portfolio.md`  
   从玩家直觉出发，设计一组彼此差异明显、合起来覆盖主要机制的代表性技能。

4. `V1A_Implementation_Sequence.md`  
   V1-A 6 个技能的逐个实施、验证、审查顺序。

5. `Skill_Mechanic_Taxonomy_Discussion.md`  
   保留讨论过程和较宽泛的分类草案。

6. `GAS_Architecture_Options.md`  
   对比不同 GAS 架构方案，并给出 V1 推荐方向。

7. `Skill_Runtime_Model.md`  
   已确认方案 C 后，拆出配置资产、通用 GA、原生机制片段、运行时流程和 V1-A 技能映射。

8. `Implementation_Roadmap.md`  
   把设计拆成 Claude 可以实施的小任务。

9. `Fireball_Minimal_Asset_Inventory.md`
   V1-A 第一个技能“火球术”的代码链路、资产复用和第一版配置建议。

本目录里的文档需要用户审核，优先使用中文。
