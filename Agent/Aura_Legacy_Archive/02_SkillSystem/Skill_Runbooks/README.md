# 技能运行链路文档约定

本目录用于存放每个代表技能的运行链路说明。

这类文档不是普通总结，而是技能实现前后的共同参照：

1. 实现前：先写“预设计版”，说明输入、目标、表现、伤害、网络、UI、失败路径大概怎么走。
2. 实现中：ClaudeCode 写代码时对照本文档，遇到偏离必须在报告里说明。
3. 审查时：Codex 对照本文档检查代码是否覆盖链路、容错、网络权威和学习注释。
4. 实现后：补成“实际运行版”，记录真实类、函数、资产和已知缺口。

每个技能文档建议包含：

1. 技能目标和当前状态。
2. 玩家视角流程。
3. 配置资产和关键字段。
4. 新增/复用类说明，包括父类和职责。
5. 从输入到触发的代码链路。
6. 目标采集、Montage、投射物、伤害 GE、ExecCalc、AttributeSet、表现/UI 的代码链路。
7. 单机、Listen Server、Dedicated Server 的网络责任划分。
8. 失败和容错路径。
9. 当前未实现或后续要补的内容。

命名建议：

```text
Skill_Runbooks/<技能名>_Runtime_Link.md
```
